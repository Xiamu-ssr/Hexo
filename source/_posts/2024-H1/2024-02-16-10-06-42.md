# 方案一、修改后端

在对应的字段上添加注解，将`Long`转为`String`后传输。

```prism language-java
	@JsonFormat(shape = JsonFormat.Shape.STRING)
    private Long payNo;
```

# 方案二、修改前端

在`js`对应的结果接收上使用`BigInt`。

```prism language-js
xxx().then((res) => {
   
    if(res){
   
        this.payNo = String(BigInt(res.payNo))
    }
```
